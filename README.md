 - Internet Explorer 6, 7, 8, 9
 - Google Chrome 14, 18
 - Mozilla Firefox 1.5, 2.0, 3.0, 3.6, 5, 9, 10, 11
 - Safari 4, 5
 - Opera 9.64, 11
 - Lynx 2.8.7 -- without progressbar :)
